## サイドメニューのデザイン・動作について

- デザインはすべて左寄せとする（折りたたみをするとアイコン＋タイトル→アイコンのみになるような位置とする）
- サイドメニューはすべて<SVGアイコン+名前>の形式で表示する。
- 折りたたみ時にはアイコンのみを表示する。
  - プロジェクトはアイコン未設定が可能。その場合は折りたたみ時にプロジェクト名の先頭2~3文字を表示する。

### 折りたたみボタン

アイコンのみ表示、クリックするたびにサイドメニューの折りたたみと展開を切り替える

### 今日、明日、今週ボタン

文言固定。
クリックしたボタンの条件に応じてタスク一覧を表示する
※いずれも時間単位は無視する（「今日」で1日の12時に表示していた場合、1日の23時59分59秒までのタスクを表示する）
今日：今日締め切り
明日：今日と明日締め切り
今週：7日以内に締め切り

### <プロジェクト名>

選択済みの名称を画面に表示、選択済みがない場合は作成日時が古いものを選択する。
アコーディオンになっていてプロジェクトのリストを折りたたみ可能
プロジェクト自体が選択された場合、タスク一覧にはそのプロジェクト内のタスクリストに該当するすべてのタスクが表示される。

### プロジェクト内タスクリストボタン

プロジェクトのリスト名を表示する。
タスクリストが選択された場合、タスク一覧にはそのリストに該当するタスクが表示される。

### ヘルプボタン

クリックするとヘルプ画面を表示する。
ヘルプ画面の詳細はまだ未定

### 設定ボタン

アイコンのみ表示、クリックするたびに表示する設定を切り替える

### アカウントボタン

ユーザーのアイコンと名前を表示する。
クリックすると下記メニューを表示する。

- ログインしていない場合
  - ログイン
  - ユーザー作成
- ログインしている場合
  - プロフィール
  - ログアウト
