# データ同期フローチャート

## ローカルファイルとクラウドストレージのための同期フロー

以下のデータが同期に使用されます。

- ローカルファイル
- ローカルからクラウドへの差分情報ファイル（ローカルの一時スペースに保存）
- クラウドからローカルへの差分情報ファイル（ローカルの一時スペースに保存）
- クラウドファイル

### 保存/アップロード中のデータ同期

1. 「ローカルファイル」を保存します。
2. 「ローカルからクラウドへの差分情報ファイル」に差分のみを保存します。
3. 「クラウドファイル」の最終更新タイムスタンプと「ローカルファイル」の最終同期タイムスタンプを比較します。
   1. 「クラウドファイル」の最終更新タイムスタンプが「ローカルファイル」の最終同期タイムスタンプより新しい場合、「ローカルファイル」の最終同期タイムスタンプ以降のデータを「クラウドからローカルへの差分情報ファイル」に保存します。
   2. ローカルからクラウドへの差分とクラウドからローカルへの差分の間に競合がある場合、ユーザーに競合を解決するよう促します。
   3. 競合が発生しない場合、または競合が解決された場合、処理を続行します。
   4. 「クラウドからローカルへの差分情報ファイル」を「ローカルファイル」に保存します。
4. 「ローカルからクラウドへの差分情報ファイル」を「クラウドファイル」に反映します。
5. 「クラウドファイル」の最終更新タイムスタンプを更新します。
6. 「ローカルファイル」の最終同期タイムスタンプを更新します。

### 定期的なデータ同期

1. 「クラウドファイル」の最終更新タイムスタンプと「ローカルファイル」の最終同期タイムスタンプを比較します。
   1. 同じ場合、以降の処理をスキップします。
2. 「ローカルファイル」の最終同期タイムスタンプ以降のデータを「クラウドからローカルへの差分情報ファイル」に保存します。
3. 「クラウドファイル」の最終同期タイムスタンプ以降のデータを「ローカルからクラウドへの差分情報ファイル」に保存します。
4. ローカルからクラウドへの差分とクラウドからローカルへの差分の間に競合がある場合、ユーザーに競合を解決するよう促します。
5. 競合が発生しない場合、または競合が解決された場合、処理を続行します。
6. 「クラウドからローカルへの差分情報ファイル」を「ローカルファイル」に保存します。
7. 「ローカルからクラウドへの差分情報ファイル」を「クラウドファイル」に反映します。
8. 「クラウドファイル」の最終更新タイムスタンプを更新します。
9. 「ローカルファイル」の最終同期タイムスタンプを更新します。
